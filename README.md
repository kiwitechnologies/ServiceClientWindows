Security-Framework
=============
## Features
- [x] HTTP Request(Post, Get, Put, Delete) implementation
- [x] Upload file on server including progress
- [x] Download file from server including progress
- [x] Cancel Http Request
- [x] Validatoin Handler


Getting started
----------------
1. Install Security Framework:
   ```bash
   
   git clone https://github.com/KiwiTechLLC/TSG-Windows.git
   ```

2. The ServiceClient Folder contains two folders: Example and ServiceClient. Example contains the sample code on how to use the ServiceClient and ServiceClient folder contains the code to generate the dll.

3. To use the dll in your project, select your application project in the Solution Explorer and then navigate to the References folder in the Solution Explorer to add a reference to it and Right-click and choose to Add Reference.  

4. Select the components you want to reference, and then click OK.
